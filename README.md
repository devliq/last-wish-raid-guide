# last-wish-raid-guide
Complete interactive guide for all 5 encounters. Track your clears, assign roles, hunt god rolls, and master the Taken mechanics of Riven's domain.

## What's included
### Encounter Guide (all 5 encounters with tabbed navigation)

- Step-by-step mechanics for Kalli, Shuro Chi, Morgeth, The Vault, and Riven

- Role assignments per encounter (Runners, Gunners, Carriers, Well, DPS lead)

- Pre-encounter checklists — click to tick off items, reset between runs

- Symbol callouts reference (Bird/Snake/Fish/Dragon + landmarks) with tooltips

- "Mark Encounter Cleared" tracking that feeds the progress bar

### Run Progress Tracker

- Live progress bar (0–100%) updating as you clear each encounter

- Header counter showing X / 5 cleared

- Color-coded encounter dots that go green when cleared

### Loot Table

- Per-encounter curated drops in a 5-column grid (Age-Old Bond → Kalli, 1KV → Riven, etc.)

- All 9 weapons with filterable cards (Solar / Arc / Void / Kinetic)

- PvE and PvP god rolls for every legendary, craftable tags, element badges

- Great Hunt armor set table for all 3 classes

### Loadouts (tabbed: DPS / Support / Runner)

- Solar DPS, Void Hunter, Arc Titan builds with weapon slots + mod recommendations

- Well of Radiance and Ward of Dawn support builds

- Shuro Chi/Vault runner and Morgeth carrier builds

**Challenges** — all 5 weekly challenges with full descriptions, clickable to track completion

**Raid Mods** — all 4 Taken mods with effects and secret chest locations

**Wish Wall** — key wishes with symbol patterns and effects

**Design** — deep void/Dreaming City aesthetic (dark purple, Ahamkara gold, Dreaming City teal), Cinzel display font, full dark/light mode toggle, fully responsive down to 375px.
